================================================================================
TEXT AUTOCOMPLETE - COMPLETE WORKFLOW DEMONSTRATION
================================================================================

This document shows the complete user workflow from start to finish.

================================================================================
STEP 1: INSTALLATION & SETUP
================================================================================

$ git clone https://github.com/eli32-vlc/text-autocomplete.git
$ cd text-autocomplete
$ pip install -r requirements.txt
$ cp config.example.json config.json
$ nano config.json  # Edit with your API key

config.json:
{
  "api_endpoint": "https://api.openai.com/v1",
  "api_key": "sk-your-actual-api-key-here",
  "model": "gpt-4",
  "max_tokens": 30,
  "temperature": 0.7,
  "pause_delay_ms": 200
}

Press Ctrl+G again to re-enable.

================================================================================
STEP 9: EXITING
================================================================================

User presses: Ctrl+X

If file modified, shows warning:
┌──────────────────────────────────────────────────────────────────────────────┤
│ Save changes? (^S to save, ^X again to exit)                                │
├──────────────────────────────────────────────────────────────────────────────┤

User presses: Ctrl+S (save) then Ctrl+X (exit)

OR

User presses: Ctrl+X again (exit without saving)

$ _
# Back to shell prompt
